Apply threshold to Optical Flow Data.

This function will remove all Optical Flow Point that is not in threshold

  • Iterate through all Optical Flow Point in Optical Flow Data
  • Compare Optical Flow Point with Optical Flow Point in previous Optical Flow Data
  • If the distance between Optical Flow Point and Optical Flow Point in previous Optical Flow Data is bigger than threshold, remove it
