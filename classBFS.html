Detailed Description

A class implementing Breadth-First Search.

Breadth-First Search (BFS) is an algorithm used for traversing or searching tree or graph data structures. In this context, the BFS class performs a Breadth-First Search on an image represented as a grid of pixels. It starts from a specified point and explores neighboring pixels in a breadthward motion, making it suitable for tasks like image segmentation, connected component analysis, and more. In this implementation, the BFS class is used to find all pixels in an image that are within a specified distance of a given color.
