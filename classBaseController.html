◆ e_CreateTempSessionData()

void BaseController::e_CreateTempSessionData ( wxEvtHandler *  parent)

Endpoint to create a copy of SessionData and store it in TempSessionData.

Use case -> User open a panel, change some data, the data is modified directly in SessionData. If the user cancel the operation, the data must be restored to the previous state. This method is used to create a copy of the current SessionData and store it in TempSessionData.
