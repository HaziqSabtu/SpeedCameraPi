Match feature descriptors in query and target images using the FLANN-\/based matcher. 

This method matches feature descriptors in the query and target images using the FLANN-\/based matcher, which is a fast approximate nearest neighbor search algorithm. The resulting matches are stored in the {\ttfamily matches} vector, where each element of the vector is a vector of {\ttfamily cv\+::\+DMatch} objects representing the matches between the query and target descriptors.
